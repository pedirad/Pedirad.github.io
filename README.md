# Pedram H. Rad — Personal Research Website

This repository contains the source code for my personal academic website:

**https://pedirad.github.io/**

The website presents my academic background, research interests, publications, projects, teaching experience, honors, and selected activities. It is designed as a minimal research profile with a visual identity inspired by chemical biology, computation, and academic CV websites.

## About

I am an undergraduate student in Pure Chemistry and Economics at Sharif University of Technology. My current interests lie at the intersection of chemistry, chemical biology, computation, logic, algorithms, and quantitative approaches to biological and social systems.
